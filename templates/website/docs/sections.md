## Universal Page Flow

Most pages follow this conversion flow:

```text
ATTENTION → TRUST → VALUE → CLARITY → OBJECTIONS → ACTION
   Hero    Social   Benefits  How it   FAQ/       CTA
           Proof             Works   Objections
```
