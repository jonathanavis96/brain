# Project: NeoQueue

Location: /home/grafe/code/NeoQueue
Purpose: Matrix-inspired desktop app for tracking discussion points to raise with manager
Tech Stack: Electron, React, TypeScript, SQLite/JSON, Zustand
Goals: Friction-free capture of "still to chat with manager" items with quick copy, follow-up threading, and completion workflow

## Description

NeoQueue is a personal comment/note management system with a Matrix dark theme. Key features:
- Left-click canvas to create new comments
- Right-click to copy comment and auto-create follow-up box (indented with connecting line)
- Completion checkbox moves items to "Completed" tab
- Matrix aesthetic with tasteful glitch effects on actions
- Global hotkey (Ctrl+Shift+Q), system tray, always-on-top option
- Auto-backup to E:\6 - Text\a - Work\Code
- Code-aware autocorrect and tab autocomplete
